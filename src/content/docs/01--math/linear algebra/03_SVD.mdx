**알고 있어야 하는 개념**
- 스칼라, 벡터, 행렬의 구분
- 항등행렬 (Identity Matrix)
- 전치행렬 (Transpose Matrix)
- 대칭행렬 (Symmetric Matrix)

**알게 될 개념**
- 고유벡터 (Eigenvector)
- 고유값 (Eigenvalue, $\lambda$ )
- 특이값 (Singular Value, $\sigma$ )
- 특이값 분해 (Singular Value Decomposition, SVD)
