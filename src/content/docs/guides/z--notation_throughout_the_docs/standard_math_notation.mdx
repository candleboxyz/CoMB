기본적인 수학 기호와 그 의미를 설명한다.

| 연산명 | 표현 기호 | 설명 | 비고 |
|--------|----------|------|------|
| 할당 | $=$ | 왼쪽 항과 오른쪽 항이 같음을 의미 | 프로그래밍의 `==`와 유사 |
| 덧셈 | $+$ | 두 수의 합 | |
| 뺄셈 | $-$ | 두 수의 차 | |
| 곱셈 | $\times$ 또는 $\cdot$ | 두 수의 곱 | 프로그래밍의 `*`와 유사 |
| 나눗셈 | $\div$ 또는 $/$ | 두 수의 몫 | |
| 거듭제곱 | $a^n$ 또는 $a^{n}$ | a의 n승 | LaTeX에서는 `a^{n}` 형식 |
| 제곱근 | $\sqrt{a}$ 또는 $a^{1/2}$ | a의 제곱근 | n제곱근은 $^{1/n}$ |
| 절댓값 | $|x|$ | x의 절댓값 | 벡터의 경우 길이(norm) |

### 합 표기법

합 표기법은 다음과 같이 사용한다:

$\sum_{i=1}^{n} a_i = a_1 + a_2 + \ldots + a_n$

여기서:
- `Σ`는 합계를 나타내는 그리스 문자 시그마(Sigma)
- i=1은 시작 인덱스
- n은 끝 인덱스
- a_i는 합할 항목

### 곱 표기법

곱 표기법은 다음과 같이 사용한다:

$\prod_{i=1}^{n} a_i = a_1 \times a_2 \times \ldots \times a_n$

여기서:
- `Π`는 곱을 나타내는 그리스 문자 파이(Pi)
- i=1은 시작 인덱스
- n은 끝 인덱스
- a_i는 곱할 항목

## 집합 표기법

집합 이론에서 사용하는 표기법과 그 의미를 설명한다.

| 연산명 | 표현 기호 | 설명 | 비고 |
|--------|----------|------|------|
| 집합 | $\{a, b, c\}$ | 원소 a, b, c를 포함하는 집합 | |
| 집합 정의 | $\{x \mid P(x)\}$ | 조건 P(x)를 만족하는 모든 x의 집합 | $\mid$는 "such that"을 의미 |
| 멤버십 | $x \in A$ | 원소 x가 집합 A에 속함 | |
| 비멤버십 | $x \notin A$ | 원소 x가 집합 A에 속하지 않음 | |
| 부분집합 | $A \subseteq B$ | A의 모든 원소가 B에도 있음 | |
| 진부분집합 | $A \subset B$ | A는 B의 부분집합이지만 $A \neq B$ | |
| 합집합 | $A \cup B$ | A 또는 B에 속하는 원소들의 집합 | |
| 교집합 | $A \cap B$ | A와 B 모두에 속하는 원소들의 집합 | |
| 차집합 | $A \setminus B$ | A에는 속하지만 B에는 속하지 않는 원소들 | |
| 대칭차 | $A \triangle B$ | $(A \setminus B) \cup (B \setminus A)$ | XOR와 유사 |
| 카디널리티 | $\lvert A \rvert$ | 집합 A의 원소 개수 | 유한 집합에만 적용 |
| 공집합 | $\emptyset$ | 원소가 없는 집합 | |
| 전체집합 | $\Omega$ 또는 $U$ | 모든 가능한 원소의 집합 | 문맥에 따라 달라짐 |
| 멱집합 | $P(A)$ | A의 모든 부분집합의 집합 | $\lvert P(A) \rvert = 2^{\lvert A \rvert}$ |
| 카르테시안 곱 | $A \times B$ | $\{(a,b) \mid a \in A, b \in B\}$ | 순서쌍의 집합 |

### 중요 집합 기호

| 기호 | 의미 | 설명 | $\LaTeX$ 문법 |
|------|------|------|------|
| $\mathbb{N}$ | 자연수 집합 | $\{1, 2, 3, ...\}$ | `\mathbb{N}` |
| $\mathbb{Z}$ | 정수 집합 | $\{..., -2, -1, 0, 1, 2, ...\}$ | `\mathbb{Z}` |
| $\mathbb{Q}$ | 유리수 집합 | 두 정수의 비로 표현 가능한 수 | `\mathbb{Q}` |
| $\mathbb{R}$ | 실수 집합 | 모든 실수 | `\mathbb{R}` |
| $\mathbb{C}$ | 복소수 집합 | 모든 복소수 | `\mathbb{C}` |

## 논리 표기법

수리 논리학에서 사용하는 표기법과 그 의미를 설명한다.

| 연산명 | 표현 기호 | 설명 | 비고 |
|--------|----------|------|------|
| 부정 | $\neg P$ 또는 $\sim P$ | P가 거짓 | |
| 논리곱(AND) | $P \wedge Q$ | P와 Q 모두 참 | |
| 논리합(OR) | $P \vee Q$ | P 또는 Q 중 하나 이상이 참 | |
| 배타적 논리합(XOR) | $P \oplus Q$ | P 또는 Q 중 정확히 하나만 참 | |
| 함의(implication) | $P \rightarrow Q$ | P이면 Q이다 | P가 거짓이면 항상 참 |
| 강한 함의 | $P \Rightarrow Q$ | P이면 Q이다 (강조) | 수학 증명에서 자주 사용 |
| 쌍조건문(iff) | $P \leftrightarrow Q$ | P이면 Q이고, Q이면 P이다 | "if and only if",<br/>"필요충분조건",<br/>"당필조건(당연하고 필요한 조건)" |
| 논리적 동치 | $P \Leftrightarrow Q$ | P와 Q가 항상 같은 진리값 | 정의에서 자주 사용 |
| 항진 명제 | $\top$ | 항상 참인 명제 | |
| 모순 명제 | $\bot$ | 항상 거짓인 명제 | |
| 전칭 양화사(for all) | $\forall x P(x)$ | 모든 x에 대해 P(x)가 참 | |
| 존재 양화사(exists) | $\exists x P(x)$ | 어떤 x에 대해 P(x)가 참 | |
| 유일 존재 | $\exists! x P(x)$ | 정확히 하나의 x만 P(x)가 참 | |

### 논리 연산자 우선순위

논리 연산자의 우선순위는 다음과 같다(높은 것부터):

1. 괄호 (), [], {}
2. 양화사 $\forall$, $\exists$
3. 부정 $\neg$
4. 논리곱 $\wedge$
5. 논리합 $\vee$
6. 함의 $\rightarrow$, $\Rightarrow$
7. 쌍조건문 $\leftrightarrow$, $\Leftrightarrow$

### 진리표

| $P$ | $Q$ | $\neg P$ | $P \wedge Q$ | $P \vee Q$ | $P \rightarrow Q$ | $P \leftrightarrow Q$ |
|---|---|----|----|----|----|----| 
| T | T | F | T | T | T | T |
| T | F | F | F | T | F | F |
| F | T | T | F | T | T | F |
| F | F | T | F | F | T | T |

## 함수와 관계

함수와 관계를 표현하는 표기법과 그 의미를 설명한다.

| 연산명 | 표현 기호 | 설명 | 비고 |
|--------|----------|------|------|
| 함수 정의 | $f: X \rightarrow Y$ | X에서 Y로 가는 함수 f | |
| 함수 적용 | $f(x)$ | x에 함수 f를 적용한 값 | |
| 합성 함수 | $(f \circ g)(x)$ 또는 $f(g(x))$ | g를 먼저 적용한 후 f를 적용 | 오른쪽에서 왼쪽으로 읽음 |
| 역함수 | $f^{-1}$ | 함수 f의 역함수 | 존재하지 않을 수도 있음 |
| 항등 함수 | $id_X$ 또는 $1_X$ | $x \mapsto x$ | |
| 치역 | $f[X]$ 또는 $Im(f)$ | $\{f(x) \mid x \in X\}$ | f의 출력 집합 |
| 역상 | $f^{-1}[Y]$ | $\{x \in X \mid f(x) \in Y\}$ | |
| 순서쌍 | $(a, b)$ | 원소 a와 b의 순서가 있는 쌍 | 순서가 중요함 |
| 관계 | $R \subseteq X \times Y$ | X와 Y 사이의 이항 관계 | |
| 함수 그래프 | $G(f)$ | $\{(x, f(x)) \mid x \in X\}$ | |

|----------|------|------|
| $\lfloor x \rfloor$ | x의 내림(floor) | 자연수 중 x보다 작거나 같은 최대값 |
| $\lceil x \rceil$ | x의 올림(ceiling) | 자연수 중 x보다 크거나 같은 최소값 |
| $\text{sgn}(x)$ | x의 부호 함수 | `-1(x<0), 0(x=0), 1(x>0)` |

## 확률과 통계

확률 및 통계에서 사용하는 표기법과 그 의미를 설명한다.

| 연산명 | 표현 기호 | 설명 | 비고 |
|--------|----------|------|------|
| 확률 | $P(A)$ | 사건 A가 발생할 확률 | $0 \leq P(A) \leq 1$ |
| 조건부 확률 | $P(A \mid B)$ | B가 주어졌을 때 A의 확률 | $= P(A \cap B)/P(B)$ |
| 결합 확률 | $P(A, B)$ 또는 $P(A \cap B)$ | A와 B가 모두 발생할 확률 | |
| 확률 분포 | $P(X=x)$ | 확률변수 X가 값 x를 가질 확률 | 이산 확률변수의 경우 |
| 확률 밀도 함수 | $f_X(x)$ | 연속 확률변수 X의 PDF | $\int f_X(x)dx = 1$ |
| 누적 분포 함수 | $F_X(x)$ | $P(X \leq x)$ | $F_X(x) = \int_{-\infty}^{x} f_X(t)dt$ |
| 기댓값 | $E[X]$ 또는 $\mu$ | 확률변수 X의 평균 | |
| 분산 | $\text{Var}(X)$ 또는 $\sigma^2$ | 확률변수 X의 분산 | $E[(X-\mu)^2]$ |
| 표준편차 | $\sigma$ | 분산의 양의 제곱근 | $\sqrt{\text{Var}(X)}$ |
| 공분산 | $\text{Cov}(X, Y)$ | X와 Y 사이의 선형 관계 | $E[(X-\mu_X)(Y-\mu_Y)]$ |
| 상관계수 | $\rho(X, Y)$ | 정규화된 공분산 | $\text{Cov}(X,Y)/(\sigma_X \cdot \sigma_Y)$ |
| 독립 | $X \perp Y$ | X와 Y가 독립 | $P(X,Y) = P(X)P(Y)$ |

### 주요 확률 분포

| 분포명 | 표기법 | 설명 | 비고 |
|-------|-------|------|------|
| 베르누이 | $X \sim \text{Bern}(p)$ | 성공 확률이 p인 이진 실험 | $P(X=1) = p$, $P(X=0) = 1-p$ |
| 이항 | $X \sim \text{Bin}(n, p)$ | n번의 독립적 베르누이 시행 | $P(X=k) = \binom{n}{k}p^k(1-p)^{n-k}$ |
| 정규(가우시안) | $X \sim N(\mu, \sigma^2)$ | 종 모양의 대칭 분포 | 많은 자연 현상에서 관찰됨 |
| 포아송 | $X \sim \text{Pois}(\lambda)$ | 단위 시간당 사건 발생 수 | $P(X=k) = e^{-\lambda}\lambda^k/k!$ |
| 균등 | $X \sim U(a, b)$ | [a, b] 구간에서 균등 분포 | $f(x) = 1/(b-a)$ for $x \in [a,b]$ |

## 미적분학

미적분학에서 사용하는 표기법과 그 의미를 설명한다.

| 연산명 | 표현 기호 | 설명 | 비고 |
|--------|----------|------|------|
| 극한 | $\lim_{x \to a} f(x)$ | x가 a에 접근할 때 f(x)의 극한값 | [극한 정의](#극한-정의) 참조 |
| 도함수/미분 | $f'(x)$ 또는 $\frac{df}{dx}$ | x에 대한 f의 미분 | 순간 변화율 |
| 편미분 | $\frac{\partial f}{\partial x}$ | 다변수 함수의 한 변수에 대한 미분 | 다른 변수는 상수로 취급 |
| 그라디언트 | $\nabla f$ | 모든 편미분을 포함하는 벡터 | $(\frac{\partial f}{\partial x_1}, ..., \frac{\partial f}{\partial x_n})$ |
| 발산 | $\nabla \cdot F$ | 벡터장의 발산 | 스칼라 필드 |
| 회전 | $\nabla \times F$ | 벡터장의 회전 | 벡터 필드 |
| 라플라시안 | $\nabla^2 f$ 또는 $\Delta f$ | 모든 2차 편미분의 합 | $\frac{\partial^2 f}{\partial x_1^2} + ... + \frac{\partial^2 f}{\partial x_n^2}$ |
| 부정적분 | $\int f(x)dx$ | f의 부정적분(원시함수) | $F(x) + C$ |
| 정적분 | $\int_{a}^{b} f(x)dx$ | a에서 b까지 f를 적분한 값 | 넓이, 부피 등 |
| 다중적분 | $\iint f(x,y)dxdy$ | 2차원 영역에서의 적분 | |
| 선적분 | $\oint_C f(x,y)ds$ | 곡선 C를 따라 적분 | |
| 테일러 급수 | $\sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!} (x-a)^n$ | a 주변에서 f의 근사 | 다항식 근사 |

### 극한 정의

극한 표기법 `lim_{x→a} f(x) = L`은 다음과 동치다:

$\forall \varepsilon > 0, \exists \delta > 0 \text{ such that } 0 < |x - a| < \delta \Rightarrow |f(x) - L| < \varepsilon$

이 정의는 "x가 a와 다르면서 a에 충분히 가까울 때마다 f(x)는 L에 충분히 가까워진다"는 것을 의미한다.

## 선형대수학

선형대수학에서 사용하는 표기법과 그 의미를 설명한다.

| 연산명 | 표현 기호 | 설명 | 비고 |
|--------|----------|------|------|
| 벡터 | $\mathbf{v}$ 또는 $\vec{v}$ | 크기와 방향을 가진 양 | 굵은 소문자로 표시 |
| 행렬 | $\mathbf{A}$ 또는 $[A]$ | 직사각형 배열의 숫자 | 굵은 대문자로 표시 |
| 스칼라 곱 | $c\mathbf{v}$ | 벡터 v에 스칼라 c를 곱함 | 방향은 같고 크기만 변함 |
| 벡터 덧셈 | $\mathbf{u} + \mathbf{v}$ | 두 벡터의 성분별 합 | |
| 내적 | $\mathbf{u} \cdot \mathbf{v}$ 또는 $\langle \mathbf{u}, \mathbf{v} \rangle$ | 두 벡터의 내적 | $\sum_i u_i v_i$ |
| 외적 | $\mathbf{u} \times \mathbf{v}$ | 두 벡터의 외적 | 3차원에서만 벡터 |
| 행렬 덧셈 | $\mathbf{A} + \mathbf{B}$ | 두 행렬의 성분별 합 | 크기가 같아야 함 |
| 행렬 곱셈 | $\mathbf{AB}$ | 두 행렬의 곱 | A의 열 수 = B의 행 수 |
| 행렬 전치 | $\mathbf{A}^T$ | A의 행과 열을 바꿈 | $(A^T)_{ij} = A_{ji}$ |
| 단위 행렬 | $\mathbf{I}$ 또는 $\mathbf{I}_n$ | 주대각선은 1, 나머지는 0 | |
| 역행렬 | $\mathbf{A}^{-1}$ | $\mathbf{AA}^{-1} = \mathbf{A}^{-1}\mathbf{A} = \mathbf{I}$ | 정방행렬만 가능 |
| 행렬식 | $\det(\mathbf{A})$ 또는 $\lvert \mathbf{A} \rvert$ | 행렬의 결정자 | 정방행렬만 가능 |
| 고유값 | $\lambda$ | $\mathbf{Av} = \lambda \mathbf{v}$를 만족하는 $\lambda$ | |
| 고유벡터 | $\mathbf{v}$ | $\mathbf{Av} = \lambda \mathbf{v}$를 만족하는 $\mathbf{v}$ | 0이 아닌 벡터 |
| 벡터 노름 | $\lVert \mathbf{v} \rVert$ | 벡터의 길이 | $\lVert \mathbf{v} \rVert_2 = \sqrt{\mathbf{v} \cdot \mathbf{v}}$ |
| 트레이스 | $\text{tr}(\mathbf{A})$ | 주대각선 원소의 합 | $\sum_i A_{ii}$ |

### 주요 행렬 유형

| 행렬 유형 | 설명 | 특성 |
|---------|------|------|
| 대각 행렬 | 주대각선 외 모든 원소가 0 | $D = \text{diag}(d_1, d_2, ..., d_n)$ |
| 대칭 행렬 | $\mathbf{A} = \mathbf{A}^T$ | 모든 i, j에 대해 $A_{ij} = A_{ji}$ |
| 직교 행렬 | $\mathbf{Q}^T \mathbf{Q} = \mathbf{QQ}^T = \mathbf{I}$ | $\mathbf{Q}^{-1} = \mathbf{Q}^T$ |
| 에르미트 행렬 | $\mathbf{A} = \mathbf{A}^H$ (A의 켤레 전치) | 복소수 행렬에서 대칭과 유사 |
| 유니터리 행렬 | $\mathbf{U}^H \mathbf{U} = \mathbf{UU}^H = \mathbf{I}$ | 복소수에서 직교와 유사 |
| 양정치 행렬 | 모든 고유값이 양수 | $\mathbf{x}^T \mathbf{A} \mathbf{x} > 0$ for $\mathbf{x} \neq \mathbf{0}$ |

## 정보이론

정보이론에서 사용하는 표기법과 그 의미를 설명한다.

| 연산명 | 표현 기호 | 설명 | 비고 |
|--------|----------|------|------|
| 엔트로피 | $H(X)$ | 확률변수 X의 불확실성 측정 | $-\sum p(x) \log p(x)$ |
| 결합 엔트로피 | $H(X, Y)$ | X, Y의 결합 불확실성 | $-\sum p(x,y) \log p(x,y)$ |
| 조건부 엔트로피 | $H(X \mid Y)$ | Y가 주어졌을 때 X의 불확실성 | $H(X,Y) - H(Y)$ |
| 상호정보량 | $I(X; Y)$ | X, Y가 공유하는 정보량 | $H(X) - H(X \mid Y)$ |
| 상대 엔트로피 | $D(P \parallel Q)$ | 두 확률분포 간의 차이 | $\sum p(x) \log(p(x)/q(x))$ |
| 크로스 엔트로피 | $H(P, Q)$ | 분포 P에 대한 Q의 부정확도 | $-\sum p(x) \log q(x)$ |
| 채널 용량 | $C$ | 채널이 전송할 수 있는 최대 정보량 | $\max I(X; Y)$ |

## 딥러닝 특수 표기법

딥러닝에서 자주 사용하는 특수 표기법과 그 의미를 설명한다.

| 연산명 | 표현 기호 | 설명 | 비고 |
|--------|----------|------|------|
| 손실 함수 | $L(\theta)$ | 모델 매개변수 $\theta$에 대한 손실 | 최소화 대상 |
| 경사 하강법 | $\theta \leftarrow \theta - \alpha \nabla L(\theta)$ | 손실을 최소화하는 매개변수 갱신 | $\alpha$는 학습률 |
| 활성화 함수 | $\sigma(z)$ | 뉴런의 출력을 비선형으로 변환 | ReLU, sigmoid 등 |
| 시그모이드 | $\sigma(z)$ | $1/(1+e^{-z})$ | 0과 1 사이의 출력 |
| ReLU | $\text{ReLU}(z)$ | $\max(0, z)$ | 음수를 0으로 변환 |
| 소프트맥스 | $\text{softmax}(z)_i$ | $e^{z_i}/\sum_j e^{z_j}$ | 확률로 정규화 |
| 역전파 | $\partial L/\partial w$ | 손실에 대한 가중치의 기울기 | 체인 규칙 적용 |
| 드롭아웃 | $p_{\text{drop}}$ | 뉴런이 비활성화될 확률 | 일반화를 위한 정규화 |
| 배치 | $B$ | 한 번에 처리되는 데이터 샘플 수 | |
| 에폭 | - | 전체 데이터셋의 한 번 완전한 학습 | |

### 딥러닝 아키텍처 표기법

| 기호 | 설명 | 비고 |
|------|------|------|
| $X$ | 입력 데이터 | 보통 형태: (배치 크기, 특성 수) |
| $y$ | 참 레이블 | 정답 또는 목표값 |
| $\hat{y}$ 또는 $y_{pred}$ | 모델의 예측 | 모델 출력 |
| $W^{[l]}$ | l번째 층의 가중치 행렬 | |
| $b^{[l]}$ | l번째 층의 편향 벡터 | |
| $a^{[l]}$ | l번째 층의 활성화 출력 | $a^{[l]} = \sigma(z^{[l]})$ |
| $z^{[l]}$ | l번째 층의 가중합 | $z^{[l]} = W^{[l]}a^{[l-1]} + b^{[l]}$ |
| $h_{\theta}(x)$ | 매개변수 $\theta$를 가진 모델의 예측 | |

## 자주 사용하는 추가 기호 및 아래/위 첨자

| 기호 | 설명 | 사용 예 |
|------|------|--------|
| $x_i$ | $i$번째 원소 또는 샘플 | $x_1, x_2, ...$ |
| $x^{(i)}$ | $i$번째 데이터 포인트/샘플 | 여러 샘플 구분 시 사용 |

## 연산자 우선순위

수학 표현식에서의 일반적인 연산자 우선순위는 다음과 같다(높은 것부터):

1. 괄호 $(\,)$, $[\:]$, $\{\,\}$
2. 함수 적용, 지수, 팩토리얼 $f(x)$, $x^n$, $n!$
3. 곱셈, 나눗셈, 모듈로 $\times$, $\div$, $\%$
4. 덧셈, 뺄셈 $+$, $-$
5. 관계 연산자 $<$, $\leq$, $>$, $\geq$, $=$, $\neq$
6. 논리 연산자 $\neg$, $\wedge$, $\vee$, $\rightarrow$, $\leftrightarrow$
